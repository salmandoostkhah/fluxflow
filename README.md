## ویژگی‌ها
- تست دانلود و آپلود (Mbps)
- پینگ، جتر، از دست رفتن پکت
- تشخیص DNS سیستم
- کشور، ISP، IP با پرچم
- نمودار عملکرد
- اکسپورت/ایمپورت اکسل
- فیلتر و جستجو
